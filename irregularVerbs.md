# Irregular verbs

## Imperfect Tense (pretérito imperfecto)

| Pronoun             | Ir     | Ver     | Ser    |
| ------------------- | ------ | ------- | ------ |
| yo                  | iba    | veía    | era    |
| tú                  | ibas   | veías   | eras   |
| él/ella/usted       | iba    | veía    | era    |
| nosotros/as         | ibamos | veíamos | eramos |
| vosotros/as         | /      | /       | /      |
| ellos/ellas/ustedes | iban   | veían   | eran   |

## Present Progressive Tense (Presente Progresivo)

Decir -> diciendo
Pedir -> pidiendo
Dormir -> durmiendo
Morir -> muriendo
Ir -> yendo
Oír -> oyendo
Traer -> trayendo
